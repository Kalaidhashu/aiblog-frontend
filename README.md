## 🤖 AI Blog Platform - Frontend

A modern AI-powered blogging platform built with React.js that enables users to create, edit, and manage blogs with AI assistance.

## 🚀 Live Demo

Frontend: https://aiblog-frontend-beta.vercel.app

---

## ✨ Features

- 🔐 User Authentication (Login/Register)
- 📝 Create, Edit & Delete Blogs
- ❤️ Like Blogs
- ⭐ Save Favorite Blogs
- 💬 Comment System
- 🤖 AI Blog Generator
- ✍️ AI Grammar Correction
- 🖼️ AI Image Generation
- 🌙 Dark / Light Theme
- 📱 Responsive UI
- 👤 User Profile
- 📊 Dashboard

---

## 🛠 Tech Stack

- React.js
- React Router DOM
- Context API
- Axios
- React Hot Toast
- TipTap Editor
- React Icons
- CSS3

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/aiblog-frontend.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm start
```

Build production

```bash
npm run build
```

## 🌐 Deployment
